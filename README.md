## Objective

This project focuses on the **exploratory analysis of global COVID-19 data** to identify trends, patterns, and key indicators throughout the pandemic.  
The objective is to transform raw public health data into **clear, structured insights** that support analytical reasoning and effective data-driven storytelling.

The analysis emphasizes:
- Trend identification over time  
- Comparative insights across regions and countries  
- Preparation of analytical datasets for visualization and reporting

## Key Business Questions Answered

This analysis addresses several analytical questions commonly encountered in business intelligence and data analytics scenarios:

- How did COVID-19 cases and deaths evolve globally over time?
- Which countries and regions experienced the highest infection rates relative to population size?
- How do mortality rates compare across countries and continents?
- What percentage of the population was infected at different stages of the pandemic?
- How did vaccination progress differ across countries when adjusted for population?
- Which regions were most impacted when comparing absolute values versus relative metrics?

These questions guide the analytical process and ensure that the insights produced are **relevant, interpretable, and actionable**.

## Project Structure & Approach

1. Data extraction and filtering using SQL  
2. Aggregation and calculation of key metrics (infection rates, death rates, vaccination coverage)  
3. Preparation of clean, visualization-ready datasets  
4. Dashboard design focused on clarity and insight delivery  

This approach mirrors real-world analytical workflows used in **business intelligence and data analytics roles**.
