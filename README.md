## Objective

This project focuses on the **exploratory analysis of global COVID-19 data** to identify trends, patterns, and key indicators throughout the pandemic.  
The objective is to transform raw public health data into **clear, structured insights** that support analytical reasoning and data-driven storytelling.

The analysis emphasizes:
- Trend identification over time  
- Comparative insights across regions  
- Preparation of data for visualization and reporting  

---

## Dataset

The project uses publicly available data provided by **Our World in Data**, covering the period from **January 1, 2020 to July 5, 2021**.
